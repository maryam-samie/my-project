# Data Analyst Final Project – Streamlit Web Application

This project is a multi-functional web application developed باستخدام **Streamlit** as part of a Data Analyst course final project.  
The application integrates **interactive games, machine learning prediction, and user interface design** in a single platform.

---

## Features

### 🎮 1. Interactive Games
The application includes multiple mini-games with user interaction and score tracking:

- Rock-Paper-Scissors  
- Memory Card Game (multi-level)  
- Typing Speed Challenge (with accuracy & scoring system)  

Each game:
- Stores user records in CSV files  
- Tracks performance and scores  
- Provides replay functionality  

---

### 2. Machine Learning Prediction
A regression model is built to predict student exam scores based on multiple factors.

#### Model Details:
- Algorithm: Random Forest Regressor  
- Preprocessing: One-Hot Encoding for categorical variables  
- Evaluation Metric: Mean Absolute Error (MAE)  

#### Input Features:
- Study hours  
- Attendance  
- Sleep hours  
- Motivation level  
- Family income  
- Teacher quality  
- And more...  

#### Output:
- Predicted exam score
